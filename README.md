## Запуск проекта

### `npm install`

Загрузив проект, установите зависимости.

### `npm start`

Запуск в режиме разработки.

### `npm run build`

Сборка приложения для продакшеша. Собранное приложение будет в папке `build`

Теперь можно деплоить!
